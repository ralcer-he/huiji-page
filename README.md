# 慧记 - AI 情绪感知日记（演示介绍页）

慧记是一款支持 AI 智能陪伴的情绪感知日记应用，支持 Windows 桌面端和 Android 移动端。

本仓库是慧记的**在线演示介绍页**，展示应用的功能截图和特性介绍。

## 在线访问

**[https://ralcer-he.github.io/huiji-page/](https://ralcer-he.github.io/huiji-page/)**

## 项目结构

```
huiji-page/
├── index.html    # 演示介绍页（含所有截图）
└── README.md
```

## 本地预览

直接双击 `index.html` 即可在浏览器中打开，无需任何依赖。

## GitHub Pages 部署

1. Fork 或创建本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，点击 Save
4. 等待部署完成即可访问
